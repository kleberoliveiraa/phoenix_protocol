h1. [HU_012] - Expor APIs REST e GraphQL para web, mobile e admin

p. *SENDO* desenvolvedor ou integrador autorizado.
*EU QUERO* consumir contratos REST e GraphQL documentados e versionados.
*PARA* integrar web, mobile, admin, academias e automacoes com seguranca e previsibilidade.

h2. Descrição

Define contratos de API com OpenAPI 3.1.1 e GraphQL, cobrindo autenticação, erros, permissoes, rate limit e versionamento.
